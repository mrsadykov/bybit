### Следующие шаги

**1. Синхронизировать ордера:**

```bash
php artisan orders:sync
```

Это обновит статусы ордеров и закроет позиции при необходимости.

**2. Проверить балансы:**

```bash
php artisan balance:check BTC --exchange=okx
php artisan balance:check ETH --exchange=okx
php artisan balance:check SOL --exchange=okx
php artisan balance:check BNB --exchange=okx
```

**3. Включить cron:**

После проверки синхронизации можно включить cron:

```bash
# Проверить crontab
crontab -l

# Если нет строки для Laravel Scheduler, добавить:
* * * * * cd /var/www/trading-bot && php artisan schedule:run >> /dev/null 2>&1
```
