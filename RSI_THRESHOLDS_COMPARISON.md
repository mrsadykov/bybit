# Сравнение порогов RSI: 40/60 vs 45/55

## Почему 40/60 лучше для реальной торговли?

### Преимущества 40/60:

1. **Меньше сделок = меньше комиссий**
   - Комиссии "съедают" прибыль
   - Меньше сделок = меньше общих комиссий
   - Больше прибыль на сделку

2. **Более качественные сигналы**
   - RSI < 40 = более сильная перепроданность
   - RSI > 60 = более сильная перекупленность
   - Более надежные точки входа/выхода

3. **Меньше ложных сигналов**
   - Строгие условия фильтруют шум
   - Меньше сделок в боковом тренде
   - Больше сделок в тренде

4. **Более стабильная торговля**
   - Меньше эмоций (меньше сделок)
   - Больше времени на анализ
   - Меньше стресса

### Недостатки 40/60:

1. **Меньше торговых возможностей**
   - Может быть недостаточно сделок для статистики
   - Может пропустить некоторые возможности

2. **Медленнее реакция**
   - Сигналы появляются позже
   - Может пропустить начало тренда

---

## Почему 45/55 лучше для бэктестинга?

### Преимущества 45/55:

1. **Больше сделок для статистики**
   - Больше данных для анализа
   - Более надежная статистика
   - Лучше видно паттерны

2. **Быстрее результаты**
   - Больше сделок за период
   - Быстрее видно результаты стратегии

3. **Больше возможностей для оптимизации**
   - Больше данных для анализа
   - Можно найти лучшие параметры

### Недостатки 45/55:

1. **Больше комиссий**
   - Больше сделок = больше комиссий
   - Может "съесть" прибыль

2. **Больше ложных сигналов**
   - Менее строгие условия
   - Больше сделок в боковом тренде

---

## Сравнение результатов

### 40/60 (консервативно):
- **Количество сделок**: 2-4 за 1000 свечей
- **Win Rate**: ~55-60% (ожидаемо)
- **Доходность**: ~0-2% (ожидаемо, меньше комиссий)
- **Качество сигналов**: Высокое
- **Комиссии**: Низкие

### 45/55 (агрессивно):
- **Количество сделок**: 4-6 за 1000 свечей
- **Win Rate**: 55.83% (проверено)
- **Доходность**: -0.05% (почти break-even, больше комиссий)
- **Качество сигналов**: Среднее
- **Комиссии**: Высокие
