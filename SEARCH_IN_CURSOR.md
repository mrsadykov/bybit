# Поиск кода в Cursor (VS Code)

## Горячие клавиши для поиска

### 1. Поиск по содержимому файлов
- **macOS**: `Cmd + Shift + F`
- **Windows/Linux**: `Ctrl + Shift + F`
- Позволяет искать текст во всех файлах проекта

### 2. Быстрое открытие файла
- **macOS**: `Cmd + P`
- **Windows/Linux**: `Ctrl + P`
- Открывает файл по имени

### 3. Поиск символов в текущем файле
- **macOS**: `Cmd + Shift + O`
- **Windows/Linux**: `Ctrl + Shift + O`
- Позволяет найти функции/классы в открытом файле

### 4. Поиск символов по всему проекту
- **macOS**: `Cmd + T`
- **Windows/Linux**: `Ctrl + T`
- Позволяет найти функции/классы во всем проекте

### 5. Command Palette (все команды)
- **macOS**: `Cmd + Shift + P`
- **Windows/Linux**: `Ctrl + Shift + P`
- Открывает панель команд

## Полезные команды в поиске

1. **"Go to Symbol"** - поиск символов
2. **"Go to Definition"** - переход к определению
3. **"Find References"** - найти все использования
4. **"Go to File"** - быстрое открытие файла

## Советы

- Используйте `Cmd/Ctrl + Click` на символе для перехода к определению
- Используйте `F12` для перехода к определению
- Используйте `Shift + F12` для поиска всех использований
- Используйте `Cmd/Ctrl + Shift + .` для поиска по всему проекту
